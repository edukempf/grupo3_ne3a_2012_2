Consultar Funcionário

Descrição: Consulta os funcionários cadastrados no sistema.

  1. Sistema exibe uma tela onde é possivel se buscar o funcionário desejado.

  2. Usuário informa o nome do cliente que deseja consultar.
  3. Para excluir um funcionário o usuário seleciona no índice e clica no botão REMOVER
  4. Para editar um funcionário, o usuário irá clicar duas vezes sobre o funcionário que deseja editar e então será exibida a tela para a edição do mesmo.


Topo

Alterar Funcionário

Descrição: Altera um funcionário cadastrado no sistema.

  1. Sistema exibe uma tela onde é possivel se informar os dados dos funcionário.

  2. Usuário informa os dados do funcionário de acordo com as alterações que deseja fazer.
  3. Para limpar os campos basta clicar sobre o botão LIMPAR onde os dados dos campos serão limpos.
  4. Para salvar a edição dos dados basta clicar no botão SALVAR
